## üé® Design Guidelines

### Color Scheme
- **Primary Background**: White or light gray
- **Accent Color**: #FFCC00 (NCDC Yellow)
- **Text**: #07080A (NCDC Black)
- **Success/Positive**: Green (#10B981)
- **Warning/Urgent**: Orange (#F59E0B)

### Typography
- **Title**: 44pt, Bold
- **Heading**: 32pt, Bold
- **Body Text**: 18-20pt, Regular
- **Footer**: 12pt, Regular

### Layout
- Keep slides clean and uncluttered
- Maximum 5-6 bullet points per slide
- Use white space effectively
- Consistent margins (1 inch all sides)

## üìã Slide-by-Slide Quick Guide

### Slide 1: Title
- Yellow background
- Black text for title
- NCDC logo (top or bottom)
- Simple, impactful

### Slide 2: The Challenge
- List current problems with icons
- Red/orange colors for problems
- Keep it brief (5 points max)

### Slide 3: The Solution
- Show LGIS interface or architecture
- Use screenshot or diagram created earlier
- Blue/green positive colors

### Slide 4: Why LGIS Wins
- 5 numbered points
- Icons for each point
- Keep each point to 2-3 words + short description

### Slide 5: Features (2x2 Grid)
- Document Review (top left)
- Revenue Dashboard (top right)
- Security (bottom left)
- Notifications (bottom right)
- Icons for each

### Slide 6: Business Impact
- Table showing before/after
- Use charts/graphs if possible
- Emphasize percentages (50%, 3x, etc.)

### Slide 7: What's Included
- Checklist format with checkmarks
- Icons for each item
- Group related items

### Slide 8: ROI
- LARGE "3-MONTH PAYBACK" callout
- Simple calculation showing math
- Green for savings, yellow for costs

### Slide 9: Comparison Table

### Slide 11: Success Metrics
- 3 sections (Month 1, 3, 6)
- Progress indicators or dashboard mockup
- Green checkmarks

### Slide 12: Risk Mitigation
- Table format
- Shield icon
- All green statuses

### Slide 13: Testimonials
- Quote boxes
- Generic industry stats
